# 🏦 Bank Management System

A complete console-based application built in **C++** to manage bank accounts and clients, developed as part of learning programming fundamentals and data structures.

---

## ✨ Features
* 📋 **Show Client List:** View all registered clients with their full details.
* ➕ **Add New Client:** Register new clients with automatic validation to prevent duplicate account numbers.
* ❌ **Delete Client:** Remove clients and clean up the database securely.
* 📝 **Update Client Info:** Modify client details such as Name, Phone, and Pin Code.
* 🔍 **Find Client:** Quickly search and locate any client using their account number.
* 💰 **Transactions Menu:**
  * 📥 **Deposit:** Add funds to a client's account balance.
  * 📤 **Withdraw:** Deduct funds with built-in validation to ensure sufficient balance.
  * 📊 **Total Balances:** View and calculate the sum of all account balances in the bank.

---

## 🛠️ Tech Stack
* **Language:** C++
* **Data Structures:** Vectors & Structs
* **Database:** File Handling (Text Files `Clients.txt`)
